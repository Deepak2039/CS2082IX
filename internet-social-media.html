5.1 Internet Basics

Welcome to the Internet & Social Media module! In this section, we'll explore how the internet works and its impact on modern communication.

Learning Objectives

  • Understand the fundamentals of how the internet works
  • Learn about different types of internet connections
  • Explore the evolution of the World Wide Web

5.1.1 How the Internet Works

What is the Internet?

The internet is a global network of interconnected computers that communicate using standardized protocols. It allows for the exchange of information and services across the world.

Key Components

  • Servers: Powerful computers that store and deliver web content
  • Routers: Direct internet traffic between networks
  • ISP (Internet Service Provider): Companies that provide internet access
  • Protocols: Rules that govern data transmission (TCP/IP, HTTP, FTP, etc.)

Data Transmission

When you access a website, your request is broken into packets that travel through multiple networks before reaching their destination. The packets may take different routes but are reassembled at the destination.

Internet vs. World Wide Web

Internet: The physical network of computers and cables
World Wide Web: A service that runs on the internet, consisting of interconnected documents and resources

5.1.2 Types of Internet Connections

Type Speed Description
Dial-up Up to 56 Kbps Uses telephone lines, slowest connection type
DSL 1-100 Mbps Uses telephone lines but faster than dial-up
Cable 10-500 Mbps Uses cable TV infrastructure
Fiber Optic 250-1000+ Mbps Uses light to transmit data, fastest option
Satellite 12-100 Mbps Good for remote areas, higher latency
5G Mobile 50-1000+ Mbps Wireless cellular network, high speed and low latency

5.1.3 Evolution of the Web

Web 1.0 (1990s)

Read-Only Web
Static websites, limited user interaction, mostly text-based content

Web 2.0 (2000s)

Read-Write Web
User-generated content, social media, web applications, rich internet applications

Web 3.0 (Emerging)

Semantic Web
AI-powered, decentralized, blockchain technology, IoT integration

Real World Impact

The evolution of the web has transformed how we live, work, and communicate. From simple text-based websites in the 1990s to today's dynamic, interactive platforms, the internet has become an essential part of daily life. Social media platforms like Facebook and Twitter, which emerged during the Web 2.0 era, have changed how we connect with others and consume information.
