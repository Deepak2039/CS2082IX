1.5 Memory (Primary and Secondary)

Learning Objectives

  • Understand the function and importance of memory in a computer system
  • Learn about different units of memory measurement
  • Compare and contrast primary and secondary memory
  • Identify different types of memory and their characteristics

1.5.1 Definition and Function of Memory

Computer memory refers to the electronic holding place for instructions and data that the computer's processor can reach quickly. It's where the computer stores data and instructions that are currently in use.

Key Functions of Memory

  • Data Storage: Temporarily holds data and instructions that the CPU needs
  • Fast Access: Provides quick access to data for the processor
  • Program Execution: Holds the program currently being executed
  • Data Processing: Stores intermediate results of processing

1.5.2 Units of Memory Measurement

Computer memory is measured in binary digits (bits) and bytes. Here are the common units of memory measurement:

Bit (b)

Smallest unit of data (0 or 1)

Nibble

1.5.3 Types of Memory: Primary and Secondary

Feature Primary Memory Secondary Memory
Volatility Volatile (RAM), Non-volatile (ROM) Non-volatile
Speed Very Fast Slower than primary
Access Directly accessible by CPU Not directly accessible
Capacity Smaller (GBs) Larger (GBs to TBs)
Cost More expensive per GB Less expensive per GB
Examples RAM, ROM, Cache HDD, SSD, Flash Drives

1.5.4 Primary Memory

Primary memory is the main memory of the computer that can be directly accessed by the CPU. It's faster but has limited storage capacity.

RAM (Random Access Memory)

  • Volatile - loses data when power is off
  • Read/Write - data can be read from and written to
  • Speed - very fast access time (nanoseconds)
  • Use - stores currently running programs and data

ROM (Read-Only Memory)

  • Non-volatile - retains data without power
  • Read-Only - data is written during manufacturing
  • Permanent - stores firmware/BIOS
  • Types - PROM, EPROM, EEPROM

Cache Memory

  • Very fast - faster than main memory
  • Expensive - more costly per MB than RAM
  • Levels - L1 (fastest), L2, L3
  • Purpose - reduces memory access time

Registers

  • Fastest - fastest memory in a computer
  • Smallest - very limited storage (few bytes)
  • Location - inside the CPU
  • Purpose - holds data being processed

1.5.5 Secondary Memory (Storage Devices)

Secondary memory provides long-term storage for data and programs not currently in use by the CPU.

Hard Disk Drive (HDD)

  • Technology - magnetic storage
  • Speed - 5,400 to 7,200 RPM
  • Capacity - 500GB to 10TB+
  • Durability - mechanical parts, less durable
  • Cost - $0.03-0.06 per GB

Solid State Drive (SSD)

  • Technology - flash memory
  • Speed - much faster than HDD
  • Capacity - 120GB to 4TB+
  • Durability - no moving parts, more durable
  • Cost - $0.08-0.20 per GB

Optical Discs (CD/DVD/Blu-ray)

  • Types - CD (700MB), DVD (4.7-17GB), Blu-ray (25-128GB)
  • Speed - slower than HDD/SSD
  • Portability - very portable
  • Durability - can be scratched/damaged
  • Use - media distribution, backups

Flash Memory

  • Types - USB drives, SD cards, microSD
  • Capacity - 1GB to 1TB+
  • Portability - highly portable
  • Durability - no moving parts, durable
  • Use - portable storage, cameras, phones

Summary

  • Memory is essential for storing data and instructions that the CPU needs to access quickly.
  • Primary memory (RAM, ROM, Cache, Registers) is faster but more expensive and volatile (except ROM).
  • Secondary memory (HDD, SSD, Optical, Flash) provides larger, non-volatile storage but is slower.
  • Memory is measured in bits and bytes, with common units being KB, MB, GB, and TB.
  • Different types of memory serve different purposes in a computer system, from ultra-fast registers to high-capacity hard drives.
