1.2 Input Devices

Learning Objectives

  • Understand the purpose and function of various input devices
  • Identify different types of input devices and their uses
  • Explain how input devices interact with computer systems

Overview of Input Devices

Input devices are hardware components that allow users to enter data and instructions into a computer. Here are some common input devices:

Mouse

A pointing device that detects two-dimensional motion relative to a surface. It typically has buttons for clicking and a scroll wheel for navigation.

Keyboard

An input device that uses an arrangement of buttons or keys to input text, numbers, and commands into a computer.

Joystick

An input device consisting of a stick that pivots on a base and reports its angle or direction to the device it is controlling.

Microphone

An input device that converts sound into an electrical signal, allowing users to input audio data into the computer.

Scanner

A device that optically scans images, printed text, handwriting, or an object, and converts it to a digital image.

Webcam

A video camera that feeds or streams an image or video in real time to or through a computer network.

Touchscreen

An input and output device that allows users to interact with a computer by touching the display screen.

Touchpad

A pointing device featuring a tactile sensor, a specialized surface that can translate the motion and position of a user's fingers.
